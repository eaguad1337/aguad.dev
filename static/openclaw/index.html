¿Qué es OpenClaw?

Tu asistente de IA que corre en TU máquina

OpenClaw es un agente de IA autónomo open-source que corre en tu computadora (Mac, Windows o Linux). Se conecta a tus apps favoritas y puede ejecutar tareas reales: manejar archivos, controlar el navegador, automatizar flujos y mucho más.

Todo con tu propia API key (Claude, OpenAI, o modelos locales vía Ollama). Tus datos nunca salen de tu máquina a menos que tú lo decidas.
